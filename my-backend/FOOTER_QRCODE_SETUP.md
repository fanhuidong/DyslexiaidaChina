# 页脚微信二维码配置指南

## 📋 配置步骤

### 1. 在 Strapi 后台添加微信二维码字段

微信二维码字段已添加到 `Global` 单页类型中，字段名为 `WechatQRCode`。

### 2. 上传二维码图片

1. 登录 Strapi 后台
2. 进入 **Content Manager** → **Single Types** → **Global**
3. 找到 **WechatQRCode** 字段
4. 点击上传按钮，选择您的微信群二维码图片
5. 填写图片的替代文本（可选，用于无障碍访问）
6. 点击 **Save** 保存
7. 点击 **Publish** 发布（重要！必须发布才能在前端显示）

### 3. 验证配置

刷新前端页面，页脚应该显示您上传的二维码图片。

## 🔍 故障排查

### 问题：二维码不显示

**可能原因**：
1. **未发布内容**：确保在 Strapi 后台点击了 **Publish** 按钮
2. **图片未上传**：检查 `WechatQRCode` 字段是否已上传图片
3. **API 连接问题**：检查浏览器控制台是否有 API 错误

**解决方法**：
1. 检查 Strapi 后台的 Global 单页是否已发布
2. 查看浏览器控制台的网络请求，确认 `/api/global` 请求是否成功
3. 检查图片 URL 是否正确

### 问题：显示"请在 Strapi 后台上传二维码"

这表示二维码字段为空或未发布。请按照上述步骤上传并发布二维码。

## 📝 字段说明

- **字段名**：`WechatQRCode`
- **类型**：Media (图片)
- **位置**：Global 单页类型
- **用途**：存储页脚显示的微信群二维码图片
